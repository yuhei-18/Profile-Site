About Me

埼玉県出身、25歳のフルスタックエンジニア。 専門学校でゲーム開発を学び、プログラミングの魅力に取り憑かれる。 ゲーム開発の道ではなく、ITに興味を持ち新卒で自動車のメーターのソフトウェア開発に携わる。 その後、自社開発ベンチャーで2年間WEBサービスの開発に携わり今に至る。 数々の新規立ち上げサービスに初期の頃から携わり、 ECサイト・大規模投票システム・WEBガチャシステム・施設予約システムの設計・開発・テストを経験する。

React

Ruby

Ruby on Rails

仕事や独学でシステムを制作する際にバックエンドで Ruby on Rails （以下 Rails）を使用していました。 Rails で API を作成しエラーハンドリングなども作成していました。 通信方法は Graphql を使用していました。REST での API を作成した経験もあります。

RSpec でテストの作成をすることもできます。

独学では、Rails のみを使用して、2ちゃんねるのようなユーザーが自由に議論するシステムを作成しました。

データベースには、主に PostgreSQL を使用していました。

Python

Django

仕事でシステムを制作する際にバックエンドで Django を使用しました。 データベースには NoSQL の MongoDB を使用しシステムの作成を行いました。 DB にデータをランダムで作成するバッチの作成や、GraphQL を使用した API の作成ができます。

施設予約システムでは、予約済みの時間帯を検索するためにビット演算を使用して処理を作成しました。
